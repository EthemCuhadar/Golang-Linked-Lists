# Stacks

* Stacks are one of the fundamental data structures in computer science.
* Working principle of stacks is LIFO (Last in first out).

![Stack Image](stack.png)

* ADT (Abstract Data Type) operations of stacks are:
* 1. Push()
* 2. Pop()
* 3. Len()
* 4. IsEmpty()
* 5. Top()
